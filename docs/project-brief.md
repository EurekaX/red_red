# 红楼梦交互式阅读网页小游戏项目简报

## 当前定位

- 项目类型：交互式阅读网页小游戏
- 题材：《红楼梦》
- 目标用户：已经喜欢《红楼梦》的读者
- 气质关键词：艺术气息、戏台梦境、可品读、可探索、有回味
- 第一章切入点：刘姥姥进大观园
- 玩家身份：梦中看客

## 核心体验

玩家进入一场“红楼梦中戏”，以梦中看客的身份在大观园中移步换景。体验借鉴橙光文字游戏的叙事选择感，但不局限于纯文字分支。玩家可以在园林地图中移动，点击人物、器物、帘幕、曲牌、环境声响等互动点，触发文本、旁白、隐藏解读、收集内容和舞台化演出。

## 美术方向

主方向：戏台梦境。

视觉应体现舞台感、灯影、帘幕、曲文、命运感和梦境层次。整体不追求现代 UI 的强功能感，而应像一出可游走的戏，一册可被玩家翻开的梦中红楼。

## 第一版空间结构

选择：园林地图式。

初版地图草图节点：

- 荣府入口：入梦、礼数、引路
- 凤姐处：寒暄、调度、赏赐
- 园门：开幕、进园
- 潇湘馆：竹影、黛玉、清冷
- 秋爽斋：探春、阔朗、书案
- 蘅芜苑：宝钗、冷香、素雅
- 宴席戏眼：笑剧、行酒令、高潮
- 醉归回廊：迷路、梦转、余韵

建议的首个 vertical slice：

- 园门
- 潇湘馆
- 宴席戏眼

## 核心循环

选路进入场景 -> 观察舞台层 -> 点击人物、器物、声响等互动点 -> 触发文本、旁白或小互动 -> 获得“梦痕”收集 -> 解锁下一处。

## 资产分类

- 场景底图
- 人物立绘或剪影
- 可点击道具
- 环境动效
- 曲文音画
- UI 题跋
- 收集卡
