# 首个 Vertical Slice 人物资产表

适用范围：第一章“刘姥姥进大观园”的首个可玩切片：园门、潇湘馆、宴席戏眼。

目标：先定义首批角色在玩法、美术和叙事中的职责，避免角色资产一开始做得过散、过满。

## 人物资产原则

- 第一版优先使用“舞台剪影 + 局部高光 + 少量关键动作”，不急着制作完整多表情 Live2D。
- 角色不是单纯装饰，应绑定至少一种互动功能：引导、观察、触发、回响或收集。
- 已经喜欢《红楼梦》的玩家不需要角色百科式介绍，文本应更像舞台旁白、人物回声和细节暗示。
- 视觉上要统一在“戏台梦境”体系中：帘幕、侧光、剪影、题跋、局部颜色象征。

## 角色 01：刘姥姥

### 角色定位

刘姥姥是第一版的情绪锚点。玩家身份是梦中看客，但很多场景会借她的外来视角打开大观园：既惊异、又机敏，既被观看，也在观看贾府。

### 出现节点

| 场景 | 出现方式 | 功能 |
| --- | --- | --- |
| 园门 | 门边剪影或入场背影 | 引导玩家进入大观园，建立外来视角 |
| 宴席戏眼 | 宴席中央动作剪影 | 触发喜剧互动和短选择 |

### 资产规格

| 资产 | 优先级 | 说明 |
| --- | --- | --- |
| 半身基础剪影 | P0 | 可用于园门和宴席 |
| 入园背影 | P0 | 园门开场使用 |
| 宴席动作姿态 A | P0 | 配合笑剧高潮 |
| 宴席动作姿态 B | P1 | 用于短选择后的回响变化 |
| 面部细节稿 | P2 | 后续精修阶段再做 |

### 互动功能

- 点击刘姥姥剪影：触发朴素、惊异但不失机敏的观察文本。
- 宴席短选择：顺着笑、装糊涂、望向贾母。
- 收集卡来源：初入园门、宴中笑声。

### 文本功能

文本不把刘姥姥写成单纯笑料。她可以承担双重视角：表层被大观园惊住，里层用乡野经验看见贾府排场的虚实。

## 角色 02：王熙凤

### 角色定位

王熙凤是宴席秩序和喜剧场面的调度者。她不是简单的 NPC，而是把“笑剧”安排上舞台的人。

### 出现节点

| 场景 | 出现方式 | 功能 |
| --- | --- | --- |
| 园门 | 声影或远处剪影 | 给玩家第一层压迫和热闹感 |
| 宴席戏眼 | 侧身剪影、袖影、眼神高光 | 触发“谁在安排笑声”的暗线 |

### 资产规格

| 资产 | 优先级 | 说明 |
| --- | --- | --- |
| 侧身剪影 | P0 | 宴席主用 |
| 袖影/手势层 | P0 | 用于调度动作 |
| 眼神高光层 | P1 | 点击后短暂显现 |
| 园门远声图标 | P2 | 可由音效和字幕替代 |

### 互动功能

- 点击凤姐：触发安排宴席、引笑、调度的旁白。
- 点击袖影：提示鸳鸯的执行动作。
- 与鸳鸯形成一组“策划者/执行者”的互动链。

### 文本功能

凤姐文本要有锋利感，不宜写成直白反派。她掌控热闹，也最能体现贾府内部精密的人情机器。

## 角色 03：鸳鸯

### 角色定位

鸳鸯是宴席互动机制的执行者。她把凤姐的调度落到具体动作上，让玩家看见笑剧怎样被推起来。

### 出现节点

| 场景 | 出现方式 | 功能 |
| --- | --- | --- |
| 宴席戏眼 | 席边立姿剪影 | 触发酒令、动作、笑声层叠 |

### 资产规格

| 资产 | 优先级 | 说明 |
| --- | --- | --- |
| 立姿剪影 | P0 | 基础呈现 |
| 手势动作层 | P0 | 指向刘姥姥或席面 |
| 低头/侧身变化 | P1 | 用于执行感 |

### 互动功能

- 点击鸳鸯：触发宴席机制说明，但要以剧情口吻呈现。
- 点击她的手势：联动酒杯、筷箸、刘姥姥动作。

### 文本功能

鸳鸯文本承担“机制解释”但不能像教程。她的存在让玩家知道宴席不是随机热闹，而是一套被安排的演出。

## 角色 04：黛玉

### 角色定位

黛玉是首个切片的审美静点。潇湘馆场景通过她建立清冷、敏感、留白的气质；宴席里她则作为笑声中的旁观者出现。

### 出现节点

| 场景 | 出现方式 | 功能 |
| --- | --- | --- |
| 潇湘馆 | 屏风后侧影、窗边低透明浮现 | 场景核心，触发细读文本 |
| 宴席戏眼 | 席间淡影或笑声层中的反应 | 群像对照，降低单纯闹剧感 |

### 资产规格

| 资产 | 优先级 | 说明 |
| --- | --- | --- |
| 屏风后侧影 | P0 | 潇湘馆核心资产 |
| 窗边淡影 | P1 | 点击竹影或诗稿后浮现 |
| 宴席淡影 | P1 | 群像中使用 |
| 诗稿残页关联图 | P1 | 与收集卡绑定 |

### 互动功能

- 点击屏风：黛玉侧影短暂浮现。
- 点击诗稿：解锁“诗稿残页”收集卡。
- 点击宴席中的黛玉淡影：触发与热闹相反的短旁白。

### 文本功能

黛玉文本要保持距离，不做长篇自述。她更像一层清冷回声，让玩家意识到大观园不是只有热闹。

## 角色 05：贾母

### 角色定位

贾母是宴席场面的中心和许可者。她的笑意味着这场笑剧可以继续，她的存在也让玩家看见贾府秩序的顶点。

### 出现节点

| 场景 | 出现方式 | 功能 |
| --- | --- | --- |
| 宴席戏眼 | 高位坐姿剪影 | 权威中心、笑声源头之一 |

### 资产规格

| 资产 | 优先级 | 说明 |
| --- | --- | --- |
| 坐姿剪影 | P1 | 宴席构图中心 |
| 灯影轮廓层 | P1 | 强调位置和权威 |
| 笑声字幕/声纹 | P2 | 可后续增强 |

### 互动功能

- 点击贾母：触发“被她看见，便被纳入热闹”的旁白。
- 短选择“望向贾母”时，触发不同回响。

### 文本功能

贾母文本应温和但有重量。她不是压迫性的反派，而是繁华秩序得以成立的中心。

## 验收标准

- 每个 P0 角色资产都能绑定明确互动，不只是摆设。
- 三个场景中角色出现节奏清楚：园门少而引导，潇湘馆静而留白，宴席密而有层次。
- 玩家能从角色互动中读出“谁在看谁、谁在安排谁、谁在被热闹吞没”。
- 第一版可以用剪影和局部动效成立，不依赖完整精细立绘。
