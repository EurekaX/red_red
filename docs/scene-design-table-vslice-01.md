# 首个 Vertical Slice 场景设计表

适用范围：第一章“刘姥姥进大观园”的首个可玩切片。

目标：用三个场景验证“园林地图式 + 戏台梦境 + 互动阅读”的核心体验，而不是一次性铺完整章。

## 设计原则

- 玩家身份是“梦中看客”，可以旁观，也可以轻度进入刘姥姥的感受。
- 玩法重点是移步移景、点击探索、舞台化文本触发，不做纯文字分支。
- 每个场景都要有可观察层、可互动层和余味层。
- 原著内容不硬讲解，优先通过人物、器物、声响、光影和旁白让玩家自己读出意味。

## 场景总览

| 场景 | 功能定位 | 玩家目标 | 核心情绪 | 产出验证点 |
| --- | --- | --- | --- | --- |
| 园门 | 入梦与进入大观园 | 理解自己是梦中看客，学会移动和点击 | 开幕、惊艳、迟疑 | 地图导航、场景进入、基础点击 |
| 潇湘馆 | 文人审美与人物气质 | 观察黛玉空间，触发细节文本 | 清冷、雅致、若即若离 | 人物/器物热点、环境文本、收集卡 |
| 宴席戏眼 | 戏剧高潮与群像互动 | 体验刘姥姥被安排成笑剧中心 | 热闹、荒诞、繁华背影 | 群像演出、节奏变化、舞台化互动 |

## 场景 01：园门

### 场景目的

建立玩家身份、空间规则和美术基调。玩家第一次意识到自己不是现代游客，而是误入一场被重新排演的红楼旧梦。

### 场景构成

| 类型 | 内容 | 说明 |
| --- | --- | --- |
| 背景 | 园门、帘幕、灯影、隐约园林轮廓 | 门不是写实景区门，应像舞台上的园门布景 |
| 角色 | 刘姥姥剪影、引路丫鬟、远处凤姐声影 | 初版可用剪影，不必完整立绘 |
| 可点物 | 门匾、灯笼、帘幕、脚边石径、远处笑声 | 每个热点触发一小段文本 |
| UI | 地图入口、返回按钮、梦痕收集提示 | 控制保持轻，不压过画面 |
| 音画 | 锣鼓轻响、帘幕开合、园中人声渐起 | 用来做“入戏”感 |

### 玩家动作

| 动作 | 反馈 |
| --- | --- |
| 点击帘幕 | 帘幕轻开，显示“梦中看客”身份旁白 |
| 点击门匾 | 触发对“大观园”作为梦场的短旁白 |
| 点击刘姥姥剪影 | 进入她的外来视角，出现朴素又惊异的观察 |
| 点击石径 | 解锁前往“潇湘馆” |

### 触发文本方向

- 不解释剧情背景，先用短句制造入梦感。
- 文本长度以 1-3 句为主，避免压成电子书。
- 可有一条“旧梦重排，繁华开场”的主旁白。

### 资产需求

| 资产 | 优先级 | 备注 |
| --- | --- | --- |
| 园门舞台背景 | P0 | 首屏气质资产 |
| 刘姥姥剪影 | P0 | 可先不做完整立绘 |
| 引路丫鬟剪影 | P1 | 负责动线提示 |
| 帘幕开合动效 | P0 | 核心入戏反馈 |
| 灯影粒子/光晕 | P1 | 氛围增强 |
| 梦痕收集卡：初入园门 | P1 | 首个收集反馈 |

### 验收标准

- 玩家能在 30 秒内理解自己可以点击场景热点。
- 玩家能从园门进入潇湘馆。
- 场景气质有戏台梦境感，不像普通古风地图。

## 场景 02：潇湘馆

### 场景目的

用一个相对安静的节点验证“细读式探索”。玩家在这里不是推进热闹剧情，而是通过空间细节读出黛玉的气质。

### 场景构成

| 类型 | 内容 | 说明 |
| --- | --- | --- |
| 背景 | 竹影、窗纱、书案、微冷灯色 | 画面应清瘦，不要堆满装饰 |
| 角色 | 黛玉侧影或屏风后身影、紫鹃剪影 | 人物可以若隐若现 |
| 可点物 | 竹影、诗稿、窗纱、香炉、屏风 | 热点以意象为主 |
| UI | 场景名题跋、热点轻提示、收集册入口 | UI 要像题签，不要像任务面板 |
| 音画 | 风过竹叶、纸页轻响、远处宴声极淡 | 形成与宴席场景的对比 |

### 玩家动作

| 动作 | 反馈 |
| --- | --- |
| 点击竹影 | 触发关于清冷、寄居、敏感的环境旁白 |
| 点击诗稿 | 解锁一张“诗稿残页”收集卡 |
| 点击屏风 | 黛玉侧影浮现，出现一句含蓄对白或内心回声 |
| 点击远处笑声 | 提示大观园另一侧的热闹，解锁宴席戏眼 |

### 触发文本方向

- 以意象触发人物气质，不做硬性人物小传。
- 可以加入“梦中看客”的观察句：看见竹影，也看见她在繁华里的孤独位置。
- 不让黛玉直接长篇自述，保持距离和留白。

### 资产需求

| 资产 | 优先级 | 备注 |
| --- | --- | --- |
| 潇湘馆舞台背景 | P0 | 第二个核心气质资产 |
| 黛玉侧影 | P0 | 可用半透明剪影 |
| 竹影动态层 | P0 | 场景识别度关键 |
| 诗稿可点击道具 | P1 | 收集卡来源 |
| 收集卡：诗稿残页 | P1 | 可用于后续图鉴 |
| 远处宴声音频 | P2 | 用于引导下一场 |

### 验收标准

- 玩家能通过 3 个以上热点理解潇湘馆的审美气质。
- 至少一个热点能产出收集内容。
- 场景节奏明显区别于宴席戏眼。

## 场景 03：宴席戏眼

### 场景目的

验证群像、节奏和戏剧高潮。这里是刘姥姥进大观园最有互动张力的一幕，玩家应感到热闹、好笑，同时隐约意识到这场笑剧背后的观看关系。

### 场景构成

| 类型 | 内容 | 说明 |
| --- | --- | --- |
| 背景 | 宴席、屏风、灯火、戏台式纵深 | 不做写实餐厅，要像一场被排练的盛宴 |
| 角色 | 刘姥姥、凤姐、鸳鸯、贾母、宝玉、黛玉、宝钗群像剪影 | 初版用层次剪影即可 |
| 可点物 | 酒杯、筷箸、席面、凤姐眼神、众人笑声、刘姥姥动作 | 形成密集互动区 |
| UI | 节奏提示、短选择按钮、梦痕收集 | 选择只影响回响，不改原著走向 |
| 音画 | 笑声层叠、杯盘声、灯影晃动、短促锣鼓 | 做出高潮感 |

### 玩家动作

| 动作 | 反馈 |
| --- | --- |
| 点击凤姐/鸳鸯 | 触发“安排笑剧”的暗线旁白 |
| 点击刘姥姥 | 触发她的应对与朴素机敏 |
| 点击众人笑声 | 笑声分层浮现，显示不同人物的观看姿态 |
| 点击酒杯/筷箸 | 触发宴席礼法和喜剧情境 |
| 做一次短选择 | 选择“顺着笑”“装糊涂”“望向贾母”等，触发不同旁白回响 |

### 触发文本方向

- 表层是热闹笑剧，里层是“谁在观看谁”。
- 选择不改写经典剧情，只改变梦中看客读到的回响。
- 文本可以更有节奏，适合短句、停顿、音效配合。

### 资产需求

| 资产 | 优先级 | 备注 |
| --- | --- | --- |
| 宴席舞台背景 | P0 | 第三个核心气质资产 |
| 刘姥姥动作剪影 | P0 | 高潮主角 |
| 凤姐/鸳鸯剪影 | P0 | 互动触发关键 |
| 贾母与众人群像层 | P1 | 氛围和观看关系 |
| 酒杯/筷箸/席面热点 | P1 | 道具互动 |
| 笑声与杯盘声音效 | P1 | 节奏关键 |
| 收集卡：宴中笑声 | P1 | 高潮奖励 |

### 验收标准

- 玩家能感觉到宴席场景比前两个节点更热闹、更密集。
- 至少有一次短选择，并能触发不同旁白回响。
- 群像关系清楚，不只是“大家在笑”的平面展示。

## Vertical Slice 总体验验收

| 维度 | 验收标准 |
| --- | --- |
| 导航 | 玩家可以从园门进入潇湘馆，再进入宴席戏眼 |
| 互动 | 三个场景合计不少于 10 个可点击热点 |
| 文本 | 文本以短触发为主，不出现长篇电子书式段落 |
| 收集 | 至少产出 3 张梦痕/收集卡 |
| 美术 | 三个场景都能看出戏台梦境气质，且情绪有区分 |
| 节奏 | 从开幕、静观到高潮，情绪递进明确 |
