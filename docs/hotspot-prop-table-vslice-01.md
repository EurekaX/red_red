# 首个 Vertical Slice 道具与热点表

适用范围：第一章“刘姥姥进大观园”的首个可玩切片：园门、潇湘馆、宴席戏眼。

目标：把玩家能点击、能观察、能触发反馈的内容拆清楚，方便策划写文本、美术画素材、前端实现交互。

## 热点设计原则

- 热点不是“可点装饰”，每个 P0/P1 热点都要承担信息、情绪、动线或收集功能。
- 点击反馈应短促明确：视觉变化 + 文本/音效/收集中的至少一种。
- 文本以 1-3 句为主，不把互动点做成电子书段落。
- 热点分布要有节奏：园门少而清楚，潇湘馆静而细，宴席戏眼密而有层次。
- 同一场景中避免所有热点反馈都一样；应有观察、回声、道具、人物、路径、收集等差异。

## 优先级定义

| 优先级 | 含义 |
| --- | --- |
| P0 | 首个可玩切片必须实现，缺失会影响核心体验 |
| P1 | 建议实现，能明显提升叙事和艺术气质 |
| P2 | 后续增强，可在第二轮迭代加入 |

## 场景 01：园门

### 热点总览

| 热点 | 类型 | 优先级 | 玩家看见什么 | 点击反馈 | 功能 |
| --- | --- | --- | --- | --- | --- |
| 帘幕 | 舞台机关 | P0 | 暗红帘幕遮住园门一角 | 帘幕轻开，灯影外泄 | 建立入戏感和身份提示 |
| 门匾 | 场景题名 | P0 | 半明半暗的“大观园”门匾 | 题跋浮现，短旁白出现 | 点明梦场 |
| 刘姥姥背影 | 角色热点 | P0 | 门边迟疑的刘姥姥剪影 | 剪影轻动，出现外来视角文本 | 情绪锚点 |
| 石径 | 路径热点 | P0 | 通向园内的弯曲石径 | 地图节点亮起，解锁潇湘馆 | 移动教学 |
| 引路丫鬟 | 角色热点 | P1 | 一名小丫鬟在帘边回身 | 手势指向园内 | 动线引导 |
| 灯笼 | 氛围道具 | P1 | 门旁灯笼微晃 | 暖光扩大，听到远处笑声 | 气氛与远处宴席伏笔 |
| 远处笑声 | 声音热点 | P2 | 画面右侧出现声纹/字幕 | 笑声略近，提示宴席方向 | 伏笔 |

### 热点详情

#### 帘幕

| 字段 | 内容 |
| --- | --- |
| 交互方式 | 点击或首次进入自动轻动，点击后完全打开一层 |
| 视觉反馈 | 暗红帘幕向两侧缓慢滑开，金色侧光进入 |
| 文本方向 | “帘子一挑，旧梦像被人重新点了灯。” |
| 音效 | 轻微布料摩擦声，极轻锣鼓点 |
| 产出 | 无收集，作为开场交互 |
| 实现备注 | 首次点击后状态持久，不重复遮挡主画面 |

#### 门匾

| 字段 | 内容 |
| --- | --- |
| 交互方式 | 点击门匾 |
| 视觉反馈 | 门匾边缘亮起，题跋 UI 浮现 |
| 文本方向 | 不解释“大观园”设定，而是提示“此处不是园林，是一场被重排的梦场” |
| 音效 | 木牌轻响 |
| 产出 | 梦痕线索：园门题跋 |
| 实现备注 | 可作为玩家首次理解“戏台梦境”的关键热点 |

#### 刘姥姥背影

| 字段 | 内容 |
| --- | --- |
| 交互方式 | 点击人物剪影 |
| 视觉反馈 | 剪影略向前，脚边出现淡光 |
| 文本方向 | 外来视角：惊异、谨慎、朴素机敏 |
| 音效 | 脚步声、衣料声 |
| 产出 | 收集卡：初入园门 |
| 实现备注 | 点击后可解锁“跟随她入园”的路径提示 |

#### 石径

| 字段 | 内容 |
| --- | --- |
| 交互方式 | 点击路径 |
| 视觉反馈 | 石径逐块亮起，地图节点“潇湘馆”点亮 |
| 文本方向 | “梦中看客不必问路，只需顺着灯影走。” |
| 音效 | 轻脚步声 |
| 产出 | 解锁下一场景 |
| 实现备注 | 这是 P0 导航热点，必须清楚易点 |

## 场景 02：潇湘馆

### 热点总览

| 热点 | 类型 | 优先级 | 玩家看见什么 | 点击反馈 | 功能 |
| --- | --- | --- | --- | --- | --- |
| 竹影 | 环境意象 | P0 | 窗外竹影落在屏风和地面 | 竹影轻摇，清冷旁白浮现 | 建立场景气质 |
| 屏风 | 人物遮挡 | P0 | 半透明屏风后有黛玉侧影 | 侧影短暂浮现又隐去 | 黛玉气质触发 |
| 诗稿 | 道具热点 | P0 | 书案上的半卷诗稿 | 纸页展开，题签出现 | 收集与细读 |
| 窗纱 | 环境道具 | P1 | 微动窗纱和冷光 | 远处宴声被压低 | 与宴席形成对照 |
| 香炉 | 道具热点 | P1 | 小香炉冒出细烟 | 烟气形成短句后散开 | 增加梦感 |
| 远处笑声 | 声音热点 | P0 | 画面边缘有淡淡声纹 | 声音变清，解锁宴席戏眼 | 动线连接 |
| 书案 | 复合道具 | P2 | 案上纸笔、砚台 | 展示更多文房细节 | 后续扩展 |

### 热点详情

#### 竹影

| 字段 | 内容 |
| --- | --- |
| 交互方式 | 点击竹影覆盖区域 |
| 视觉反馈 | 竹影轻摇，冷色光略增强 |
| 文本方向 | 通过竹影暗示清冷、寄居、敏感，而不是直接写人物介绍 |
| 音效 | 风过竹叶 |
| 产出 | 无或梦痕线索：竹影 |
| 实现备注 | 竹影是潇湘馆识别度最高的环境热点 |

#### 屏风

| 字段 | 内容 |
| --- | --- |
| 交互方式 | 点击屏风 |
| 视觉反馈 | 屏风后黛玉侧影浮现 1-2 秒 |
| 文本方向 | 含蓄对白或梦中看客旁白，保持距离和留白 |
| 音效 | 纸屏轻响 |
| 产出 | 解锁黛玉人物回声 |
| 实现备注 | 不做长对话，避免破坏“看客”身份 |

#### 诗稿

| --- | --- |
| 交互方式 | 点击书案上的诗稿 |
| 视觉反馈 | 诗稿局部展开，墨迹微亮 |
| 文本方向 | 以残页、题句、旁批感呈现，不做全文展示 |
| 音效 | 翻纸声 |
| 产出 | 收集卡：诗稿残页 |
| 实现备注 | 首个细读类收集热点 |

#### 远处笑声

| 字段 | 内容 |
| --- | --- |

## 场景 03：宴席戏眼

### 热点总览

| 热点 | 类型 | 优先级 | 玩家看见什么 | 点击反馈 | 功能 |

#### 短选择按钮

| 字段 | 内容 |
| --- | --- |
| 交互方式 | 在触发刘姥姥/鸳鸯链路后出现 3 个选项 |
| 视觉反馈 | 选项像戏牌或题签，不做现代按钮感 |
| 文本方向 | 选项：顺着笑、装糊涂、望向贾母 |

| 产出 | 三种不同旁白回响，剧情走向不变 |
| 实现备注 | 验证“像橙光但不止文游”的选择回响机制 |

## 验收标准

- 玩家在每个场景至少能发现 3 个有效热点。
- P0 热点全部有视觉反馈、文本反馈和明确功能。
- 三个场景的热点节奏有明显差异：园门教学、潇湘馆细读、宴席戏眼高潮。
- 至少 3 个热点产出收集物，至少 1 个热点触发短选择。
- 热点文案不使用现代任务系统口吻，保持戏台梦境和红楼语境。
